== README

Dead simple rails app to try out websocket rails.

== Get Started
* git clone the repo to your local system
* run `bundle install`
* to get the server running, run: `puma -C config/puma.rb`
* open a browser to: `http://localhost:9292`
* open up the javascript console
* type something into the text field and hit Send

== Notes
* note that the console has items coming in and the list echoes back the input text.
